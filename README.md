# 엄준식
급식을 알려주는 엄준식입니다.

## 명령어
```
$고딩급식 학교이름 날짜(일)

$고딩석식 학교이름 날짜(일)

$중딩급식 학교이름 날짜(일)

$중딩석식 학교이름 날짜(일)

$초딩급식 학교이름 날짜(일)


$ping

$link
```
